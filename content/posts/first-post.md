### 1. 代码高亮
作为一个技术博客，清晰的代码展示是必须的。这里展示一段经典的 Python 递归：

```python
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

print("Hello, World!")
````
